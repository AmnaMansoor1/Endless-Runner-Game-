# Endless Runner Game

This Unity project is an endless runner game developed to learn and implement fundamental game development concepts.

## Key Features

- **Infinite Platform Generation**: Platforms are dynamically instantiated along the Z-axis, creating an endless running track for the player.
- **Random Obstacle Spawning**: Obstacles spawn randomly on the platforms, adding variety and challenge to the gameplay.
- **Collision Detection**: Trigger colliders are utilized to detect when the player passes through or collides with objects.

## Purpose

This was my very first Unity project, created to understand the initial concepts of game development, including object instantiation, prefab usage, basic physics, and collision handling.
